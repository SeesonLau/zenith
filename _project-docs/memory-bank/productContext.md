# Product Context — Zenith

## Why It Exists
Most productivity apps focus on one domain (habit trackers, budgeting apps, journals). Zenith combines all four into one dark-themed personal dashboard so there's one place to review your day.

## Problems It Solves
- Habit tracking apps don't have finance or diary
- Finance apps don't connect to how you're spending your time
- Diary apps are siloed from productivity data
- Most apps don't work offline

## Who Uses It
A single user — the developer/owner. Not a SaaS product. Personal tool.

## Core User Flows
1. **Habits**: Start a timer on a habit → timer runs → stop → session logged
2. **Finance**: Add a transaction (amount, category, date) → view summary + analytics
3. **Diary**: Write an entry → attach mood + images → browse calendar view
4. **Leisure**: Log a leisure session (type, title, duration) → view history

## Design Philosophy
- Dark-first UI (slate-950 background)
- Offline-first data (WatermelonDB local)
- Simple, no clutter — each screen does one thing
